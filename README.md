# 今天又活了一天 - StillAlive

> 原来你还活着啊

一款以「生存确认」和「记忆沉淀」为核心的轻量化打卡工具。

## 想法

活着，本身就值得庆祝。

每天早上或晚上，打个卡，告诉自己：**恭喜你又活过了一天**。

不需要写长篇大论，不需要精心排版，甚至可以什么都不记录——因为这只是一个打卡工具。但如果你愿意，可以记录今天有意义的事，可以为重要的人写一份「人物小传」。

我认为人活着最重要的东西就是记忆，在离开之前我们能想起的其实也只有回忆，所以为自己重要的人或事去记录，这个写下来的过程，本身就是加深记忆的过程，为自己增加回忆的节点。

### 核心功能

- **生存打卡** - 每日确认"我还活着"
- **意义记录** - 记录今天发生的有意义的事
- **人物小传** - 为重要的人建立情感档案，记录印象、经历、生日
- **死亡确认** - 连续多日未打卡时，向紧急联系人发送确认邮件

### 设计理念

- 轻量化，不强求记录
- 温和的文案，不制造焦虑
- 通过大事记和人物小传，创造与世界和现实的联系

## 技术栈

```
StillAlive/
├── apps/
│   ├── mobile/        # React Native (Expo)
│   ├── miniprogram/   # 微信小程序 (Taro)
│   ├── web/           # Web 端
│   └── server/        # 后端服务
└── packages/          # 共享包
```

- **Monorepo**: Turborepo + pnpm
- **移动端**: React Native + Expo
- **小程序**: Taro
- **语言**: TypeScript

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发
pnpm dev:mobile      # 移动端
pnpm dev:miniprogram # 小程序
pnpm dev:web         # Web端
pnpm dev:server      # 后端
```
